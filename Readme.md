# Kata09

URL: http://codekata.com/kata/kata09-back-to-the-checkout/

## Usage

Install dependencies:

```bash
yarn
```

Calculate with sample rules:

```bash
yarn ts-node src/index.ts AAABBD
```

Run tests:

```bash
yarn vitest
```
